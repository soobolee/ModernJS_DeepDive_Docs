
# Math 

표준 빌트인 객체 Math

셍상지 함수가 아니다.
따라서, Math는 정적 프로퍼티와 정젝 메서드를 제공한다.

## Math 프로퍼티

1.Math.PI
  원주율 값 3.141592653589793을 반환한다.

## Math 메서드

1. Math.abs
2. Math.round
3. Math.ceil
4. Math.floor
5. Math.sqrt
6. Math.random
7. Math.pow
8. Math.max
9. Math.min
